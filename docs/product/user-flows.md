# User Flows (Draft)

Primary flows
- New customer creates a custom order and is directed to Orders
- Customer reviews design, approves/rejects, then pays before production
- Customer purchases standard products via cart + checkout
- Customer reorders from Orders page
- Admin manages all orders in Admin Orders dashboard
- Admin uploads/replaces design and requests revisions
- Admin updates order status and shipping info
- Admin impersonates customer order view for troubleshooting
