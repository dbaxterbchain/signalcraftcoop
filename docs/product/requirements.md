# Product Requirements

Scope
- Custom orders with design review workflow
- Store orders with NFC options
- Customer Orders page for self-service actions (approve, pay, reorder)
- Store cart + checkout flow for standard products
- Admin Orders dashboard for all orders, status updates, and design approval
- Admin impersonation tool for customer order view

Non-goals (for now)
- Marketplace for other vendors
- Subscription billing
- Live payment integration (Stripe) until LLC/TPT is complete

Open questions
